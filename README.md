# hh-bot

Telegram-бот для поиска вакансий коммерческого директора в нефтянке на HH.ru.

## Возможности

- 🔍 Поиск вакансий по ключевым словам и городам
- 🤖 AI-фильтрация через DeepSeek
- 📊 Статистика и логирование
- ⏰ Авто-поиск 2 раза в день (9:00 и 18:00 UTC)
- 🌐 Webhook режим (не засыпает на Render)

## Команды

| Команда | Описание |
|---------|----------|
| `/start` | Старт бота |
| `/search` | Поиск вакансий сейчас |
| `/schedule on/off` | Включить/выключить авто-поиск |
| `/stats` | Статистика |
| `/filters` | Текущие фильтры |
| `/salary 250000` | Изменить мин. зарплату |
| `/relocate` | Список городов |
| `/cleanup` | Очистить старые записи |
| `/help` | Справка |

## Переменные окружения

| Переменная | Описание | Обязательная |
|------------|----------|--------------|
| `TELEGRAM_TOKEN` | Токен бота от @BotFather | ✅ |
| `CHAT_ID` | ID чата для отправки вакансий | ✅ |
| `DEEPSEEK_API_KEY` | API ключ DeepSeek | ❌ |
| `SCRAPERAPI_KEY` | Ключ ScraperAPI для обхода блокировок | ❌ |
| `RENDER_EXTERNAL_URL` | URL сервиса на Render (авто) | ❌ |

## Деплой на Render

1. Создайте Web Service на Render
2. Подключите GitHub репозиторий
3. Установите переменные окружения
4. Render автоматически установит `RENDER_EXTERNAL_URL`
5. Бот запустится в режиме webhook и не будет засыпать
